# robotics

This repo contains the Damson hexapod project (based on the Freenove Hexapod Kit) and related references.

- Product requirements: `Damson/PRD.md`
- Engineering notes (approaches & learnings): `Damson/ENGINEERING_NOTES.md`

Quick start (PlatformIO):

```pwsh
pio run
pio run -t upload
pio device monitor -b 115200
```
